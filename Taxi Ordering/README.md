# Прогнозирование количества заказов такси на следующий час

## Описание проекта
Разработка системы предсказания объема заказа.

## Навыки и инструменты
- Python
- Pandas
- Scikit-learn
- statsmodels

## Общий вывод
Построена модель для предсказания количество заказов такси на следующий час на основе исторических данных о заказах такси в аэропортах.
